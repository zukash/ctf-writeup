# Cryptordle

## 概要

* 6回以内に `[x, y, z, w, v]` を当てる
* 予測 `[a, b, c, d, e]` に対して `(x - a) * (y - b) * (z - c) * (w - d) * (v - e) % 31` が返る

## 観察

* 31 は素数なので `x / (x - 1)` から `x` を一意に特定できる

## 解法

* `[0, 0, 0, 0, 0]` を投げて `x * y * z * w * v` を得る
* `[1, 0, 0, 0, 0]` を投げて `(x - 1) * y * z * w * v` を得る
* `x / (x - 1)` から `x` を得る
